## BOX TO BOX (E-Commerce-app)
An online shopping cart
[Link to the website](https://boxtobox.website/)

## Key Features:
* Secured Payment gateways: Razorpay
* Offers and Coupons
* Wallet and wallet history
* Chart & Graph reports on Admin side
* Sales Reports
* Referral 

## Technologies used:
* Node js
* Express js
* MongoDB
* Bootstrap
* Nginx

## Services used:
* AWS
